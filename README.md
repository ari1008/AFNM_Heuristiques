### **Application**

## **Lancement de l'application**

Pour démarrer l'application, suivez les étapes ci-dessous :

1. Placez-vous à la racine du projet.
2. Lancez la commande suivante pour démarrer les services avec Docker Compose :

   ```bash
   docker compose up -d
   ```
